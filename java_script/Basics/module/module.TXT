
MODULE IN JS IS A FILE THAT CONTAINS JS CODE AND ALLOWS YOU TO EXPORT PART OF IT SO THEN CAN BEUSED IN OTHER FILES

IT HELPS ORGANIZED CODE , REUSE FUNCTIONALITY AND AVOOID NAME COMPLEX BY KEEPING VARIABLES SCOPED TO THE MODULE


THERE ARE 2 TYPES


1.IMPORT
2.EXPORTS

1.IMPORT:
=========
WE HAVE NAMED IMPORT AND DEFAULT IMPORT

BY USING THIS WE CAN USE THE LOGIC FROM OTHER FILE WHICH IS EXPORTED BY USING EXPORT
 
WE HAVE 2 TYPES OF IMPORT

1.NAMED IMPORT : WHILE IMPORTING THE FILE WE HAVE TO PROVIDE THE SAME NAME WITH WHICH IT IS DECLARE OR EXPORTED

2.DEFAULT IMPORT: BY USING THIS WE CAN IMPORT LOGIC BY GIVING ANY NAME 



2.EXPORT:
=========
WE HAVE NAMED EXPORT AND DEFAULT EXPORT

EXPORT: BY USING THIS WE CAN SEND THE LOGIC TO OTHER FILE 

WE HAVE 2 TYPES OF EXPORT

1.NAMED EXPORT : IN A FILE WE CAN HAVE MANY NAMED EXPORT

2.DEFAULT EXPORT:IN A FILE WE CAN HAVE ONLY ONE DEFAULT EXPORT
