DOM : (DOCUMENT OBJECT MODEL):
==============================

==> IT IS THE PROGRAMMING INTERFACE FOR WEB DOCUMENT
==> IT SHOWS THE ELEMENT PRESENT IN HTML IN A TREE LIKE STRUCTURE.
==> IT IS USED TO DELETE , ADD ,UPDATE THE ELEMENS PRESENT IN HTML
==> WHENEVER WE WILL CHANGE ANY NODE IT WILL IN THE RELOADING OF THE ENTAIRE PAGE/WEBPAGE.

==================

    2. .innerText

    6. .id
